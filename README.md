# 📦 Inventory Management System (IMS)

An intuitive browser-based **Inventory Management System (IMS)** that helps you **track stock, vendors, usage, and key inventory metrics** in one place.

---

## ✨ Key Features  

✅ **User Access** – Secure sign-up, login, or anonymous access  

📊 **Dashboard Insights** – At-a-glance view of:  
- Total stock items  
- Current inventory value  
- Unique categories  
- Low-stock alerts  
- Visual stock distribution charts  

📦 **Stock Management** –  
- Add new items with category, vendor, units, price & threshold  
- Quick add for frequently used items  
- Track usage and monitor balances  
- Edit stock details anytime  

📑 **Reporting** –  
- Used stock reports with filters  
- Export stock & usage data as CSV  

🤝 **Vendor Management** –  
- Add, view, and update vendor details  
- Keep track of suppliers for smooth restocking  

---

## 🖥️ Modules / Pages  

- **Dashboard** → Key stats, charts, and low-stock alerts  
- **Stock Details** → Full inventory with filtering + CSV export  
- **Add Stock** → Manual entry & Quick Add shortcuts  
- **Use Stock** → Record consumption & track recent usage  
- **Used Stock Report** → Detailed report of consumed stock  
- **Vendors** → Vendor management (add/edit/list)  
- **Authentication** → Login, Signup, Guest access  

---

## 🛠️ Tech Stack  

- **Frontend**: HTML, CSS, JavaScript (TailwindCSS for UI)  
- **Backend**: Firebase (optional)  
- **Database**: Firebase / Local Storage  

---

## 🚀 How to Use  

1. Open the system in your browser.  
2. Sign up / log in or continue as a guest.  
3. Add stock items with required details.  
4. Track usage, monitor balances, and check low-stock alerts.  
5. Manage vendor information for smooth reordering.  
6. Export reports anytime for analysis.
